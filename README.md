# Zayssystem
Create discord bots in secs

## Install here
```
  npm i zayssystem OR npm i zayclient@1.0.1
  
```
or go to current release

# How to use ZayJS
```
const {ZayJS} = require('zayssystem');
const client = new ZayJS();
client.get(url);
client.post(url,options = { form: {}})
client.create(data={id,token})
```
